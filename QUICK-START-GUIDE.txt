================================================================================
TROUBLESHOOTING: "Cannot run script" Error
================================================================================

If you get an error about "cannot be loaded" or "execution policy":

1. Press Win + X
2. Select "Windows PowerShell (Admin)" or "Terminal (Admin)"
3. Copy and paste this command:

   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

4. Press Enter, then type Y and press Enter again
5. Close PowerShell
6. Try running the script again
